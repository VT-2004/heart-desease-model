About Our Project

Project Overview

This project is an end-to-end heart disease prediction system that leverages machine learning to assess a patient's risk of developing heart disease based on various health parameters.

Developed as a demonstration of machine learning application in healthcare, the system provides a user-friendly interface for inputting health data and receiving an immediate, multi-model prediction report.

Machine Learning Algorithms Used

Our system employs a suite of diverse machine learning algorithms to provide comprehensive predictions. Each model brings a unique approach to pattern recognition, contributing to a more robust assessment:

  • Logistic Regression: A foundational statistical model for binary classification, known for its interpretability.
  • K-Nearest Neighbors (KNN): A non-parametric, instance-based learning algorithm that classifies based on the majority class of its nearest neighbors.
  • Support Vector Machine (SVM): A powerful model that finds an optimal hyperplane to separate data points into classes.
  • Decision Tree: A tree-like model that makes decisions based on feature values, easy to visualize and understand.
  • Random Forest: An ensemble method that builds multiple decision trees and combines their outputs for improved accuracy and robustness. This model generally showed the highest accuracy in our evaluations.

Technical Stack

The system is built using a modern Python-based stack:

  • Machine Learning: Scikit-learn, Pandas, NumPy
  • Web Framework: Flask
  • User Interface: HTML5, CSS3
  • Development Environment: Google Colab (for ML training), Local Python Environment (for Flask app)

Our Goal

Our primary goal is to showcase the potential of machine learning in assisting preliminary risk assessment for heart disease. While this tool provides data-driven predictions, it is designed for educational and demonstrative purposes only.
